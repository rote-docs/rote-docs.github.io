Credits

Without the incredible open-source community, this theme wouldn’t be what it is today. Here are some of the projects that were used in Rote.
Hugo Of course, a Hugo theme would be nothing without Hugo itself!
UIKit A lightweight and modular front-end framework for developing fast and powerful web interfaces.
Mermaid A library for rendering all kinds of charts and diagrams on the web.
MathJax A JavaScript display engine for mathematics that works in all browsers. No more setup for readers. It just works.
Elasticlunr.js Lightweight full-text search engine in Javascript for browser search and offline search.
